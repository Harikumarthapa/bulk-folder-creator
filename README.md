## About the Project
Bulk Folder Creator is a free, privacy-first web utility that allows users to instantly generate up to 10,000 nested directories in their browser. 

Manual directory creation introduces inconsistencies into professional workflows. This utility eliminates manual folder creation, guaranteeing identical structures across every new project, repository, or client archive. 

All processing happens locally within the user's web browser. Directory structures and text inputs are never uploaded, stored, or transmitted to an external server.

## Core Capabilities
* **3 Input Methods:** Type folder names, import a CSV/spreadsheet, or define a bulk number sequence.
* **Live Validation:** The tool automatically filters duplicates and skips OS-restricted characters (`\ / : * ? " < > |`).
* **Deep Nesting:** Supports up to 20 levels of nested subfolders.
* **Cross-Platform Exports:** Generate directory trees natively for your operating system.

## Supported Export Formats
* **ZIP Archive** (Universal download)
* **Windows Batch Script** (`.bat`)
* **macOS / Linux Shell Script** (`.sh`)
* **PowerShell Command** (Copy to clipboard)

## Folder Structure Templates by Use Case
We built this tool to standardize workflows across multiple industries:
* **Web Development:** `Project_Name / src, public, assets, tests, docs`
* **Photography & Videography:** `Client_Name / raw, edited, final-exports, contracts`
* **Business & Agencies:** `Client_Name / q1-reports, invoices, meeting-notes, assets`
* **Business & Agencies:** `Education & Course Creators: Course_Name / week-01, assignments, video-lessons, resources`
